# NotificationType

The type of notification to mark as seen. If not provided, all notifications will be marked as seen.

## Enum

* `Follows` (value: `'follows'`)

* `Recasts` (value: `'recasts'`)

* `Likes` (value: `'likes'`)

* `Mentions` (value: `'mentions'`)

* `Replies` (value: `'replies'`)

* `Quotes` (value: `'quotes'`)
